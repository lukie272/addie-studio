# ADDIE Studio

ADDIE Studio is a prototype clinical education course builder.

Type a training idea, optionally add learner and organisation details, then the app searches OpenAlex for published evidence and drafts an ADDIE-informed training package.

The generated package downloads as a ZIP folder of separate printable documents.

## GitHub Pages

Enable with: Settings -> Pages -> Deploy from branch -> main -> root

Expected site URL: https://lukie272.github.io/addie-studio/

## Safety note

This is a prototype. Clinical content, evidence currency, scope of practice, local policy alignment, and patient-safety controls must be reviewed by an appropriate clinician, educator, or subject matter expert before use.
